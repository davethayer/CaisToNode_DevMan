Optional Parameters

  • FacilitySubmittalComment An optional comment provided in the CERS facility XML submittal.
  • PointSubmittalComment An optional comment provided in the CERS point emissions XML submittal.
  • An excluded plants spreadsheet containing IDs for plants which would otherwise be included as individual point sources, These plants will be reported as ONP - operating but included in the non-point inventory.
    • ExcludedPlantsWorkbook The filename of the spreadsheet containing the excluded plants spreadsheet.
    • ExcludedPlantsWorksheet The name of the worksheet tab containing the plant list.
    • ExcludedPlantsCountyFipsColumn The column containing the county FIPS codes.
    • ExcludedPlantsPlantIdColumn The column containing the plant IDs. If the plant IDs include the county code (e.g. an ICIS ID) this should be the same as the county FIPS column above.
    • ExcludedPlantsStartingRow The first row containing plant ID data. This allows skipping header rows. Note that county and plant IDs are validated using regular expressions, so headings would likely be ignored anyway.
  • An included plants spreadsheet containing IDs for plants which should be included in the submittal. If the submittal is limited to type "A" major sources these plants are included in additional to those calculated as major during processing. This allows ensuring that non-type "A" plants previously reported as type "A" can be accounted for. If the submittal type is not limited to type "A" sources (3-year cycle) the submittal will be limited to the plants in the spreadsheet.
    • IncludedPlantsWorkbook The filename of the spreadsheet containing the excluded plants spreadsheet.
    • IncludedPlantsWorksheet The name of the worksheet tab containing the plant list.
    • IncludedPlantsCountyFipsColumn The column containing the county FIPS codes.
