CAIS to EIS Identifier Cross Reference

EIS Identifier Base CAIS IDs Format Notes
FacilityIdentification.FacilitySiteIdentifier County FIPS (Plant.PLA_A_CNTY): ccc
Plant Number (Plant.PLA_A_PLNT): ffff		 CO00000008ccc0ffff This format is required by EPA for ICIS-AIR compatibility.

APCD staff generally use a shorter "AIRS ID" (ccc-ffff) format.
ReleasePointIdentification.Identifier County FIPS (Plant.PLA_A_CNTY): ccc
Plant Number (Plant.PLA_A_PLNT): ffff
Stack Number (stacks.stk_afsid): sss		 ccc-ffff-Ssss The release point identifier includes the letter 'S' to differentiate it from the unit identifier.
UnitIdentification.Identifier County FIPS (Plant.PLA_A_CNTY): ccc
Plant Number (Plant.PLA_A_PLNT): ffff
Point Number (Point.POI_A_PNT): uuu		 ccc-ffff-uuu
ProcessIdentification.Identifier County FIPS (Plant.PLA_A_CNTY): ccc
Plant Number (Plant.PLA_A_PLNT): ffff
Point Number (Point.POI_A_PNT): uuu
Process number (process.pro_afsid): pp		 ccc-ffff-uuu-pp
ReleasePointApportionmentIdentification.Identifier See ProcessIdentification.Identifier ccc-ffff-uuu-pp Same as ProcessIdentification.Identifier
ReleasePointApportionmentPathIdentification.Identifier See ProcessIdentification.Identifier ccc-ffff-uuu-pp Same as ProcessIdentification.Identifier
SiteControlIdentification.Identifier ccc
Plant Number (Plant.PLA_A_PLNT): ffff
Point Number (Point.POI_A_PNT): uuu
Process number (process.pro_afsid): pp
Control type code (pr_pol.prp_priceq, pr_pol.prp_secceq): ttt		 ccc-ffff-uuu-pp-ttt In CAIS the control device is not a separate record type, it is treated as attribute to the process-pollutant record, which may have primary and secondary control codes.
ControlPathDefinitionControlIdentification.Identifier See SiteControlIdentification.Identifier Same as SiteControlIdentification.Identifier
ControlPathIdentification.Identifier See ProcessIdentification.Identifier ccc-ffff-uuu-pp Same as ProcessIdentification.Identifier

In CAIS, the identifiers are unique within the containing entity, For example, the same plant number could occur in multiple counties, and the same process number could occur for processes under multiple units within a plant. Since CAIS generates IDs sequentially, this is a common occurrence.

EIS identifiers have been constructed so that the identifier is unique within the State's inventory.
