Supported CERS Facility Inventory Elements

CERSDataType Elements

Identification information regarding the submission file.

Element Name Description
UserIdentifier Unique identifier of a user record. This identifier is assigned by the receiving system and is unique for each user. Permissions for updating data are granted based on the user identification. (Required)
ProgramSystemCode The code that represents the information management system which has responsibility for the data in a linked or interrelated information management system. (Required)
EmissionsYear The year of the submitted emissions.
SubmittalComment Any comments regarding the file submission.

FacilityNAICS Elements

Contain the North American Industry Classification System code assigned to facility site based on economic profile.

Element Name Description
FacilityNAICS Complex data type for North American Industry Classification System code assigned to facility site based on economic profile.
NAICSCode Code defining the NAICS designation for the facility

FacilityIdentification Elements

Define identifiers by which the facility site is known or has been known, and the system associated with the identifier.

Element Name Description
FacilitySiteIdentifier Identifier for the facility with the defined Program System Code (Required)
ProgramSystemCode Code defining the organization associated with the FacilityIdentifier (Required)
StateAndCountyFIPSCode Federal Information Processing Standards code defining the county (or county equivalent) location of the facility

AddressDataType Elements

Define the location where a facility site or organization is located or an individual may be reached.

Element Name Description
LocationAddressText The physical location of a facility site or organization.
SupplementalLocationText The text that provides additional information about a place, including a building name with its secondary unit and number, an industrial park name, an installation name, or descriptive text where no formal address is available.
LocalityName The name of the city, town, village, or other locality.
LocationAddressStateCode The alphabetic codes that represent the name of the principal administrative subdivision of the United States, Canada, or Mexico.
LocationAddressPostalCode The code that represents a U.S. ZIP code or International postal code.

GeographicCoordinatesDataType Elements

Identify the geographic location of the facility site, emission release point, or event.

Element Name Description
FacilitySiteGeographicCoordinates Complex data type which identifies the geographic location of the facility site, emission release point, or event.
LatitudeMeasure The measure of the angular distance on a meridian north or south of the equator. (Required)
LongitudeMeasure The measure of the angular distance on a meridian east or west of the prime meridian. (Required)

Element Name Description

Element Name Description

UnitDesignCapacityUnitofMeasureCode Unit of measure for the design capacity of the emissions unit.

ReleasePoint Elements

Identify the point at which emissions are released into the environment, via a stack or fugitive release.

Element Name Description
ReleasePointTypeCode Code that identifies the type of release point.
ReleasePointStackHeightMeasure The height of the stack from the ground.
ReleasePointStackHeightUnitofMeasureCode The stack height unit of measure
ReleasePointStackDiameterMeasure The internal diameter of the stack (measured in feet) at the release height.
ReleasePointStackDiameterUnitofMeasureCode The stack diameter unit of measure.
ReleasePointExitGasVelocityMeasure The velocity of an exit gas stream.
ReleasePointExitGasVelocityUnitofMeasureCode The unit of measure for the velocity of an exit gas stream value.
ReleasePointExitGasFlowRateMeasure The value of the stack gas flow rate.
ReleasePointExitGasFlowRateUnitofMeasureCode The unit of measure for the stack gas flow rate value.
ReleasePointExitGasTemperatureMeasure The temperature of an exit gas stream (measured in degrees Fahrenheit).
ReleasePointFugitiveHeightMeasure The fugitive release height above terrain of fugitive emissions.
ReleasePointFugitiveHeightUnitofMeasureCode The fugitive release height unit of measure.
ReleasePointStatusCode Code that identifies the operating status of the release point.
ReleasePointStatusCodeYear The year in which the release point status became applicable.
