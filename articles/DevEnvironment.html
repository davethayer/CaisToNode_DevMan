Development Environment.

CaisToNodeCore was written in C# 10 under the .NET 6 framework.

NuGet Package Dependencies

  • ClosedXML provides Excel file reading and writing functionality. Used by Reporter to generate report files and by CaisManager to load included and excluded plants lists.
  • CommunityToolkit.Mvvm provides the messaging service used for Issue and LogMessage messages which can be consumed by the GUI and the Reporter
  • Config.Net provides access to configuration JSON files and persistent storage of preferences. The preferences are stored in appsettings.json and accessed via the SettingsManager object's CaisValues, EisValues, NaaParameters, and Settings properties.
  • Dapper is used by CaisManager and NodeManager[see note] to access SQL server databases.
  • Microsoft.Data.SqlClient is used to manage SQL Server connections.
  • Microsoft.VisualBasic provides CSV parsing functionality for SnapshotManager.
  • StyleCop.Analyzers (Optional) helps with code sanitation.
