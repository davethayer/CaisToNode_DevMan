Configuration Settings

Configuration is specified in appsettings.json. These entries may be overridden in a user settings file specified by the Bootstrap.UserSettingsFileName key. The user settings file is located in the user settings folder specified by the Bootstrap.UserSettingsFolderName key. The user settings folder is located under the user's APPDATA folder.
