Processing Steps for Emissions

Emissions processing is handled using an object implementing IPointEmissionsProcessor. This object depends upon the completion of the facility inventory steps for removal of unneeded records and data validation.

Operating on process pollutant records in the downloaded CAIS process pollutant table:

  1. Populate node facade tables.
    • Generate container records as needed. These will be separate from the ones in the facility inventory.
      • FacilitySiteRecords.
      • FacilityIdentificationRecords.
      • EmissionsUnitRecords.
      • UnitIdentificationRecords.
      • ProcessRecords.
      • ProcessIdentificationRecords.
      • ReportingPeriodRecords for each process.
    • ReportingPeriodEmissionsRecords for each pollutant.
