Mandatory Parameters

Prior to extracting data the following parameters must be set in SettingsManager.Settings, typically using an interactive GUI:

  • InventoryYear The NEI inventory year.
  • IncludeAllCategories Indicates whether this is a type A major source-only submittal (annual) or if all source categories are included (3-year cycle). Potential type A sources are determined automatically based upon emissions.
  • AuthorName is included in the EIS submittal. The included source can be expanded or limited further using optional spreadsheets (see below.)
  • UserIdentifier The email address of the submitter.
  • WorkingDirectory The directory where the output directory will be located. It typically contains the downloaded snapshot and CSV lookup files.
  • EisSnapshotXmlFileName The state-wide Facility Snapshot XML file downloaded from the EIS gateway. This may be left in a zip file as downloaded or extracted to a raw XML file if disk space is not an issue, but there's no significant performance gain from doing so.
  • EisControlMeasureLookupCsvFileName The full path to the Control Measure Code CSV table downloaded from the EIS gateway.
  • EisNaicsLookupCsvFileName The full path to the NAICS Code CSV table downloaded from the EIS gateway.
  • EisSccLookupCsvFileName The full path to the Source Classification Code - EIS CSV table downloaded from the EIS gateway.
  • EisUnitTypeLookupCsvFileName The full path to the Unit Type Code CSV table downloaded from the EIS gateway.

Note that it is possible to deal with minor changes in an EPA CSV file's layout by modifying the appropriate settings in the EisConstants.CodeTableParsing section of appsettings.json.
