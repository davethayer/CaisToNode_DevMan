Processing Steps for Emissions Units

Emissions unit (points in CAIS) processing is handled using an object implementing IEmissionsUnitProcessor.

Operating on emissions unit records in the downloaded CAIS release point table.

  1. Remove unneeded records, i.e. those having zero emissions or belonging to plants not included in the current submittal.
  2. Validate data to ensure that it meets EIS standards. Value range limits and defaults are configurable.
  3. Set operating status to Operating and status code year to null since any other units will have been removed.
  4. Convert CAIS design rates to those used in EIS. The conversion factors are configurable.
  5. Generate closure records for units belonging to facilities included in this submittal which are not operating in CAIS but are shown as operating in the current EIS snapshot.
  6. Populate node facade tables.
